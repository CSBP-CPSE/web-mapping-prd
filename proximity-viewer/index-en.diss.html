How to use this application

Click and drag the map using the mouse to navigate the map. Use the mouse wheel to zoom the map in or out.

Map navigation options are also provided on the left side of the map :

  • To go to full screen mode
  • To zoom the map in
  • To zoom the map out
  • To return the map to the initial extent (Canada)
  • To select another indicator to show on the map
  • To navigate to a census metropolitan area

The legend contains the information required to interpret the map. The values assigned to dissemination blocks are normalized indices of proximity. The dissemination block color on the map is assigned according to the values contained in the legend. Color classes are determined by quintile, each class contains a fifth of the data.

A slider bar provides control over the opacity of the dissemination block layer.
